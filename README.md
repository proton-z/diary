## 本地任务目标网站（Node.js + SQLite）

### 你将获得什么

- **新增任务**：包含「任务标题」「目标（更重要）」「标签」
- **编辑任务**：新增后可随时编辑标题/目标/标签
- **未完成视图**：默认仅显示未完成任务
- **标签**：内置默认标签 **长期 / 短期**，也支持自定义标签；可按某个标签筛选任务
- **本地存储**：数据保存在 `backend/data/tasks.db`

### 启动

在项目根目录运行：

```bash
npm install
npm run dev
```

然后打开：`http://localhost:5178`

### 使用提示

- 右上角「新增任务」创建
- 列表中「编辑」可修改任务
- 左侧点击标签可筛选（再次点击同一标签可取消）
- 勾选/取消「仅看未完成」可切换显示范围
